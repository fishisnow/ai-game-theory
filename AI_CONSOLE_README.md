# AI 思考控制台功能说明

## 功能概述

AI思考控制台是一个实时显示AI思考过程和决策的窗口，让您能够深入了解AI在博弈过程中的策略分析和决策逻辑。

## 主要功能

### 1. 实时日志显示
- **思考过程** 🤔：显示AI分析当前局势的思考过程
- **决策记录** ✅：记录AI的最终选择和决策理由
- **错误信息** ❌：显示API调用失败或其他错误
- **系统信息** ℹ️：记录游戏开始、回合进展、结果等系统事件
- **警告信息** ⚠️：显示异常情况和警告

### 2. 控制台操作
- **自动滚动**：自动滚动到最新日志，可手动关闭
- **最小化/最大化**：支持窗口最小化，不影响游戏进行
- **清空日志**：一键清空所有历史日志
- **导出日志**：将日志导出为文本文件，便于分析

### 3. 日志分类和样式
- 不同类型的日志使用不同颜色和图标区分
- 显示精确的时间戳（精确到毫秒）
- 显示AI名称和厂商信息
- 支持结构化数据显示（JSON格式）

## 使用方法

### 打开控制台
1. 点击右下角的绿色终端图标 🖥️
2. 图标上的红色数字显示当前日志数量
3. 控制台会在游戏开始时自动打开

### 查看AI思考过程
1. 开始AI竞赛后，控制台会实时显示：
   - 游戏开始信息
   - 每轮开始的参与者
   - AI的思考分析过程
   - AI的决策和选择
   - 回合结果和淘汰信息
   - 游戏结束和获胜者

### 日志类型说明

| 类型 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| thinking | 🤔 | 蓝色 | AI的思考和分析过程 |
| decision | ✅ | 绿色 | AI的最终决策和选择 |
| error | ❌ | 红色 | 错误信息和异常 |
| info | ℹ️ | 灰色 | 系统信息和游戏进展 |
| warning | ⚠️ | 黄色 | 警告信息 |

## 技术特性

### 性能优化
- 虚拟滚动支持大量日志
- 智能自动滚动检测
- 内存友好的日志管理

### 数据导出
- 支持导出为文本文件
- 包含完整的时间戳和结构化数据
- 便于后续分析和研究

### 响应式设计
- 适配不同屏幕尺寸
- 支持最小化模式
- 不干扰主界面操作

## 注意事项

1. **隐私保护**：日志可能包含API密钥等敏感信息，导出时请注意保护
2. **性能影响**：大量日志可能影响浏览器性能，建议定期清空
3. **网络延迟**：AI响应时间取决于网络状况和API服务器性能
4. **错误处理**：API调用失败时会自动使用备选策略，确保游戏继续
